To create a game, select game->create on menu.

After that, the screen will show next.

Just fill in the data and the game will be created.

To start a game match, go to game->all games.

After that, the games will be shown. To start a game, just the button on the right that says "New match"

Then you can play the game.

While you are playing, on the landmark, the dialog will be shown.

To save game session, just click "Save". (NOTE! If you loaded game session or if you are already saved it, the program will ask you weather to save as new or owerwrite existing) When you save game, it may improve your experience and rank so you can enjoy more benefits that our program has.
